The goal of Kakuro is to fill all blank squares in the grid with digits from 1 to 9 so that the sum of the numbers in each entry matches the clue associated with it. No digit may be repeated within an entry.

Select a blank square and input a number (1-9). Make sure that the number does not repeat within the horizontal or vertical clue sum it belongs to.

Use known combinations of numbers for sums. For example, the only way to sum 3 using two numbers is 1+2. Also, keep track of possible values per cell as you go.

The game ends when all cells are filled correctly according to the clues. If every sum is matched and all rules are followed — you win!
